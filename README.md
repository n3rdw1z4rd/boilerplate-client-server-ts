# boilerplate-client-server-ts
boiler plate for typescript client/server projects

## server-dependencies:
* typescript: `^5.2.2`
* express: `^4.18.1`
* ws: `^8.8.1`

## client-dependancies:
* typescript: `^5.2.2`
* vite: `^5.3.1`
* preact: `^10.22.0`
* lz-string: `^1.5.0`

client also uses a simple and minimal CSS reset by Andy Bell:
https://piccalil.li/blog/a-more-modern-css-reset
